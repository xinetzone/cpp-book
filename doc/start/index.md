# 快速上手 C++

```{epigraph}
学习之前，需要明确你的需求，不要为了学习而学习，而是为了解决需求而学习。

学习某项技能或者某个概念，需要对构成它的逻辑和设定有一些认知，并不断地强化这些认知，才会学习地高效。

-- xinetzone
```

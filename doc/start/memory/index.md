# 映射到硬件

```{hint}
C++ 提供到硬件的直接映射。当使用一个基本运算时，其具体实现就是硬件提供的，通常是单一机器运算。例如，整数加法 `x+y` 就是执行一条整数加法机器指令。

C++ 将 **机器内存** 看作一个内存位置序列，可在其中存放（有类型的）**对象** 并可使用 **指针** 寻址。

**指针** 在内存中表示为一个机器 **地址**。

基本语言结构到硬件的简单映射对原始的底层性能至关重要。C 和 C++ 的基本机器模型是基于计算机硬件而非某种形式的数学。
```

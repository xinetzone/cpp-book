C++ 基本概念¶

C++ 是一种高级编程语言，设计了大量数据结构和指令。

查看 术语表 了解一些基本概念。

C++ 通过操作计算机内存实现各种各样的任务。先看看一些术语：

单元

一个完整东西的与众不同的（如命名）部分。

位

计算机中基本信息单位。可能的取值为 0 或者 1。

值

一组二进制 位，具体的含义由其 类型 决定。

类型

定义了一组可能的值以及一组（对象上）的操作。

对象

存放某种类型值的内存空间。

地址

一个值，用来在计算机内存中查找一个对象。

字节

大多数计算机中的基本寻址单元。1 byte = 8 bit。

字

计算机的基本单元，通常用于保存一个整数的单元。1 word = 4 byte

初始化

为对象赋予最初的（初始）值。

变量

给定类型的命名对象；除非没有初始化，否则包含一个值。

声明

声明是一条语句，负责为程序引入一个新的名字，并指定该命名实体的类型。

定义

一个实体的声明，提供了所有必要信息，令程序可使用此实体。一种更简单的说法：分配了内存的声明。

函数

一个命名的代码单元的，可以从程序中其他部分调用它；计算的逻辑单元。

数据

计算中用到的值。

类

用户自定义类型，可以包含数据成员、函数成员和成员类型。

接口

一个声明或一组声明，指明了一段代码（如一个函数或类）如何被调用。

迭代

重复执行一段代码的动作。

文件

计算机中永久保存信息的容器。

头文件

一个包含声明的文件，其中的声明用于程序中不同部分共享接口。

基本类型与声明变量¶

C++ 提供了若干基本类型，例如：

  • bool：布尔类型。可取 true 或 false。

  • char：字符类型。如 'a'，'\n' 等

  • int：整数类型。如 1，42 等

  • double：双精度浮点数类型。如 2.3，5.0

某种基本类型都与硬件特性直接相关，其尺寸固定不变，决定了其中所能存储的值的范围：

sizeof(bool) // 1 个字节

1

sizeof(char) // 1 个字节

1

sizeof(int) //  4 个字节

4

sizeof(double)  // 8 个字节

8

声明语句，可以这样编写：

int 苹果数量;

其中 苹果数量 便是一个变量。变量的命名规则见 标识符 或者 名字。

标识符¶

标识符 是以字母和下划线开头，后接 0 个或者多个（大写或者小写）字母、数字或者下划线的序列。

int hi_world;
int HiWorld;

int Hi World; // 错误：因为标识符不能有空格

双下划线¶

以下划线开头或者包含连续两个下划线的标识符是为 C++ 实现预留的，你的程序中不要使用这种标识符。例如：

int _foo; // 不要这样用
int foo_boor; // 正确
int foo__bar; // 不要这样用
int foo_; // 正确

关键字¶

关键字 是 C++ 用来表示语言结构的特殊标识符。

基本运算符¶

运算符可以用于操作类型。下面仅仅介绍一些基本运算符：

算术运算符
x+y   // 加法
+x   // 一元加法
x-y  // 减法
-x   // 一元减法
x*y  // 乘法
x/y  // 除法
x%y  // 取模
比较运算符
x==y   // 相等
x!=y  // 不相等
x<y  // 小于
x>y  // 大于
x<=y  // 小于等于
x>=y  // 大于等于
赋值运算符

=

为一个变量赋予新的值。

初始化¶

通用初始化方法是使用花括号内的一组初始化器列表：

double d1 {2.3};
d1

2.3000000

当然也可以使用传统的方法，即赋值语句：

double d1 = 2.3;
d1

2.3000000

类型转换¶

赋值运算、算术运算以及初始化会在基本类型间进行有意义的类型转换。

double d = 2.2; // 初始化浮点数
d

2.2000000

int t = 7; // 初始化整数
t

7

d = d + t; // 将求和结果赋值给 d
d // 可以看出此时 d 是浮点数

9.2000000

t = d * t; // 把乘积结果赋值给 t
t

64

d * t 是一个 double 值，而 t 是 int，所以最终的结果被截断为整数。
